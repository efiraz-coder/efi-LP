# מערכת ניהול תוכן - Admin Panel

## גישה למערכת

### כניסה למערכת הניהול
1. גש לכתובת: `http://localhost:5173/admin/login`
2. הזן את הסיסמה: **admin2026**
3. לחץ על "כניסה"

### שינוי הסיסמה
לשינוי הסיסמה, ערוך את הקובץ:
```
src/context/AuthContext.jsx
```

שנה את השורה:
```javascript
const ADMIN_PASSWORD = 'admin2026'
```

## תכונות המערכת

### 1. עריכת תוכן
המערכת מאפשרת עריכה של כל התוכן באתר:
- **Hero Section** - כותרות, תיאורים, כפתורי CTA, תמונות
- **Bio Section** - ציטוטים, פסקאות, שנות ניסיון
- **Router Section** - שירותים, כותרות, תיאורים
- **Footer** - זכויות יוצרים וקישורים

### 2. שמירת שינויים
- לחץ על כפתור "שמור שינויים" בראש הדף
- השינויים נשמרים ב-localStorage של הדפדפן
- השינויים יישארו גם לאחר רענון הדף

### 3. איפוס תוכן
- לחץ על כפתור האיפוס (⟲) בראש הדף
- מאפס את כל התוכן למצב ברירת המחדל

### 4. צפייה באתר
- לחץ על "צפה באתר" לפתיחת האתר בטאב חדש
- השינויים יופיעו באתר מיד לאחר השמירה

## הערות חשובות

### אחסון מקומי
- התוכן נשמר ב-localStorage של הדפדפן
- השינויים לא נשמרים בשרת
- ניקוי cache של הדפדפן ימחק את השינויים

### לייצור (Production)
למערכת ייצור מומלץ:
1. להחליף את מערכת האימות בפתרון מאובטח יותר
2. לשמור את התוכן בשרת/מסד נתונים
3. להוסיף הצפנה לסיסמאות
4. להוסיף אימות דו-שלבי

## פתרון בעיות

### לא מצליח להתחבר
- ודא שהסיסמה נכונה: `admin2026`
- נקה את ה-localStorage: `localStorage.clear()`
- רענן את הדף

### השינויים לא נשמרים
- ודא שלחצת על "שמור שינויים"
- בדוק את ה-console לשגיאות
- ודא ש-localStorage פעיל בדפדפן

### השינויים לא מופיעים באתר
- רענן את דף האתר
- ודא שהשינויים נשמרו בלוח הבקרה
- בדוק שאין שגיאות ב-console
